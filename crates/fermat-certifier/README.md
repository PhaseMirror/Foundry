# fermat-certifier-rs: Exact Algebraic Cycle Verification

Integer-exact verification of multiplicity states and algebraic cycles.

## Features
- **ExactCertifier**: Cycle validation using snapping to integer snapshots.
- **Structure Validation**: Enforces strict dimensional constraints for $G_{NS}$ and $H$.
- **Auditable Records**: Generates `AlgebraicCertificate` and `EffectiveCertificate` structures.
