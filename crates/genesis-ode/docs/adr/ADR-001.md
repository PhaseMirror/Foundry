## Context
The Genesis ecosystem requires a single, stable, numerically bounded reference engine for substrate-indexed persistence simulation. All Lane A engineering, Exploder stress testing, Builder assembly, tether metrics, and governance review must operate against this immutable reference without altering its semantics, ODE form, or claim boundaries.

## Decision
We instantiate Track A ODE Core as the sole canonical scalar engine. It consists exactly of:

1. **Substrate-indexed persistence state** $C_X(t)$
2. **The generalized ODE:**
   $$\frac{dC_X}{dt} = \alpha(C_X^* - C_X) + \beta G_X - \gamma S_{\rm eff,X}(t) + \eta A_X(t) + \kappa \sum w_{ij}(C_j - C_i) - \delta_X$$
3. **Impedance / drag bridge:**
   $$\rho_X = C_X^* (1 - e^{-\lambda S_{\rm eff,X}})$$
   $$\Omega_X = \frac{1}{\sqrt{1 - (\rho_X / C_X^*)^2}}$$
   $$D_{k,X} = \Omega_X - 1$$
4. **Metallurgical fatigue prototype** (cyclic $S_{\rm eff,Met}(t) = S_0 \sin(\omega t)$) as first validated substrate
5. **Explicit validation contracts**, telemetry export, and reproducible Euler (or higher-order) integration under fixed seeds
6. **Evidence tiers** restricted to:
   - [E] (externally validated)
   - [S] (supported by simulation + calibration)
   - [I] (internal exploratory)

### Core invariants (non-negotiable):
- No prime-indexed dynamics inside the scalar core (prime structure is metadata only).
- Dynamics live on $C_X(t)$ (or future microdomain $C_{X,i}(t)$ only via superseding ADR).
- Core may receive bug fixes, numerical improvements, and backwards-compatible telemetry — never structural mutation from experimental branches.
- Every output artifact carries `schema_version`, `run_id`, `source_adr_ids`, `tier`, and `provenance`.

## Consequences
- **Positive:** Provides a rock-solid, replayable baseline for Exploder/Builder tether. Enables deterministic metallurgical runs and τ computation.
- **Negative:** Requires explicit governance overhead for any future microdomain lift.
- **Tradeoff:** Exploration lives in Lane A Exploder sandbox; core remains protected.
