# ADR-017: Phase 1 – Formalizing the Formula of Universal Law (FUL)

**Status**: Proposed

**Context**
- We have a precise, step‑by‑step algorithm for Kant’s Formula of Universal Law.
- The algorithm was applied to the classic “lying‑to‑the‑murderer” maxim, yielding a contradiction in conception and a perfect‑duty prohibition.

**Decision**
- Adopt the **FUL checklist** as a reusable procedural artifact for any Kantian analysis within the project.
- Encode the checklist in documentation and reference it in all future ethical‑decision modules.

**Consequences**
- Provides a clear, formal method for testing the universalizability of maxims.
- Guarantees that any future moral‑logic component can invoke a verified procedure rather than an ad‑hoc reasoning process.
- Sets the stage for systematic cross‑checking with the other two formulations.
