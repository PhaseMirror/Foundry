# ADR-019: Phase 2 – Formalizing the Formula of Humanity (End‑in‑Itself)

**Status**: Proposed

**Context**
- A parallel algorithmic checklist for Kant’s Formula of Humanity has been drafted.
- The checklist isolates the maxim, identifies the treatment relation, checks for simultaneous treatment as an end, and yields a verdict.
- It was applied to the lying‑to‑the‑murderer maxim, producing a perfect‑duty prohibition.

**Decision**
- Adopt the **Humanity checklist** as a formal, reusable artifact for any Kantian analysis in the project.
- Document the checklist in the ADR repository and reference it from the ethical‑decision modules.

**Consequences**
- Guarantees that future moral‑logic components evaluate the “means‑as‑mere‑means” condition consistently.
- Enables cross‑checking with the Universal‑Law and Kingdom‑of‑Ends procedures for robust validation.
- Provides a clear foundation for extending Kantian reasoning to new domains (AI alignment, governance, etc.).
