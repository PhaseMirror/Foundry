# mcp-server-rs: Multiplicity Control Protocol Server

High-performance Rust implementation of the Model Context Protocol (MCP) server for the Multiplicity substrate.

## Features
- **MCP Protocol**: Standardized tool registry, tool discovery, and tool execution.
- **Axum Integration**: High-performance HTTP/JSON-RPC transport layer.
- **Generic Tool Trait**: Extensible architecture for registering new tools dynamically.
- **Safety**: Fully type-safe tool execution.

## Usage
### Build
```bash
cargo build --release
```

### Test
```bash
cargo test
```
