# multiplicity-cli: Multiplicity Command-Line Interface

High-performance Rust CLI for the Multiplicity substrate, providing governance, audit, and diagnostic tools.

## Features
- **Initialization**: Project setup with configurable templates.
- **Dissonance Analysis**: Command-line interface for running automated consistency checks.
- **L0 Validation**: Validate system invariants against the Ξ-Constitution.
- **Drift Detection**: Automated drift auditing.

## Usage
### Build
```bash
cargo build --release
```

### Test
```bash
cargo test
```
