## Standard ADR Format
Each ADR in this directory adheres to the following structure:
- **Title**: The title of the ADR.
- **Status**: Proposed, Accepted, Rejected, or Superseded.
- **Context**: The forces at play, including technological, mathematical, and research considerations.
- **Decision**: The specific architectural choice made.
- **Consequences**: The resulting context, including trade-offs and future constraints.
