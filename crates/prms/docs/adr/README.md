# PRMS Architecture Decision Records

This directory contains the Architectural Decision Records (ADRs) for the Prime-Recursive Multiplicity Substrate (PRMS).

## Process

1. Create a new ADR in `proposed/` using the [template](templates/TEMPLATE.md).
2. Discuss and review the ADR.
3. Once a decision is reached, move the ADR to `accepted/`, `rejected/`, or `superseded/`.
