\section{ERE Score}
\begin{equation}
  \operatorname{ERE}(q) = \frac{1}{5} \sum_{k=1}^{5} \mathbf{1}[P_k(q) = \text{fail}] \in [0, 1]
\end{equation}
Score $0.0$ = all passes clean (METATRON: YES). Score $1.0$ = all five failed.
