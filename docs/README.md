# Prime / Docs

This directory serves as the centralized knowledge base for the Prime Materia Commons and the Multiplicity Sovereign Core framework. 

## Overview

Here you will find all foundational theory, architectural decision records (ADRs), policy documents, and whitepapers that dictate the design and operation of the Phase Mirror and Multiplicity systems.

- **Whitepapers & Theory:**
  - `WHITEPAPER.md` and `The_Foundry_Whitepaper.md`
  - `Multiplicity_Theory.md`
  - `M_Quantum_Structures.md`
  - `QAI-HTS-LANGLANDS.md`
- **Architectural Decision Records (ADR):**

- **Constitutional Governance:**

- **Agent Guidelines & Protocols:**

  - `Lambda_Proof_Binding.md`
  - `Prime Move Sequence Audit Macro.md`
- **Security:**
  - `SECURITY.md` and `SECURITY_ADVISORIES.md`
- **Phases & Next Steps:**
  - Instructions for Phase progression (`PHASE5-02-START-HERE.txt`, `PHASE6_NEXT_STEPS.md`, etc.)

All modifications to the architecture or governance must be drafted here as an ADR and ratified according to the Sedona Spine.
